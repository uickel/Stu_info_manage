# 简易学生信息管理系统（控制台版）
Python控制台版学生信息管理系统，基于面向对象封装，支持添加/查询/修改/删除学生信息，包含异常处理。

## 核心功能
1. 添加学生：姓名/学号/年龄，学号查重，年龄范围校验（0-120）
2. 查询学生：按学号查询，无结果提示
3. 修改年龄：按学号修改，校验新年龄合法性
4. 删除学生：按学号删除，无结果提示
5. 异常处理：捕获非数字输入、范围错误等，程序不崩溃

## 技术要点
- 面向对象：StudentManager类封装所有功能
- 数据存储：列表+字典存储学生信息
- 异常处理：捕获ValueError，处理输入错误
- 循环逻辑：while循环实现系统持续运行

## 运行方式
1. 本地安装Python3.x
2. 直接运行student_manager.py文件：
   ```bash
   python student_manager.py
